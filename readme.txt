== Description ==

This Plugin provide you options to manage the stock quantity automatic increase throughout daily, weekly, monthly, hourly and yearly schedule type options of all your woocommerce products, scheduling stock manage with automatic increasing stock quantity. 
You will have the flexibility to schedule stock manage for both variation and simple product by schedule type daily, monthly with increasing stock quantity. 


== SCHEDULE STOCK MANAGE ==

WooCommerce Schedule Stock Manager plugin gives you option to set schedule stock manage that provide option to set schedule type and stock quantity.

== SCHEDULE TYPE ==

WooCommerce Schedule Stock Manager plugin gives you the option to set schedule type for stock quantity which lets your customers ensure about stock available for that specific product. 

== STOCK QUANTITY ==

WooCommerce Schedule Stock Manager plugin gives you the option to increase stock quantity base on the schedule type. 

== SCHEDULE QUANTITY TYPE (PRO) ==

WooCommerce Schedule Stock Manager plugin allows you to add or update schedule stock quantity to stock quantity.

==  Free Plugin Features == 
 
* Set up Schedule Stock Manage to add stock quantity with specific schedule type 
* Schedule Stock Manage for both variation and simple product
* Schedule Type with options Every Minute, Hourly, Monthly, Twice a day, Weekly, Twice Monthly to set stock quantity 
* Variation Product and Simple Product with different stock type and stock quantity 
* Manually have option to schedule stock manage product wise 
* Manually Enable, Disable schedule stock manage

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After Plugin Active go to Products-> Add new -> Inventory (tab).
